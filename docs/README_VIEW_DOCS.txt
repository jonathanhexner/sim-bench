HOW TO VIEW DOCUMENTATION
=========================

OPTION 1: Use Preview in Cursor (RECOMMENDED)
----------------------------------------------
1. Open any .md file in Cursor
2. Press Ctrl+Shift+V
3. See formatted preview!

If you get "null" error, try:
- Close and reopen Cursor
- Right-click the file -> "Reopen Editor With..." -> "Text Editor"
- Then try Ctrl+Shift+V again


OPTION 2: Web Server (For HTML Viewer)
---------------------------------------
1. Double-click "serve_docs.bat"
2. Open browser to: http://localhost:8000/VIEW_DOCS.html
3. Click documents in sidebar
4. Press Ctrl+C in the command window to stop


OPTION 3: Read as Plain Text
-----------------------------
Just open the .md files directly - they're readable even without formatting!


TROUBLESHOOTING
---------------
- "null" error in Cursor: Restart Cursor
- HTML viewer doesn't load: Use Option 2 (serve_docs.bat)
- Python not found: Use Option 1 or 3
